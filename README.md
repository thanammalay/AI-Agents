# 🤖 AI LinkedIn Agent

AI LinkedIn Agent is a Python-based multi-agent system designed to automate LinkedIn content creation. It uses AI models to generate posts, summaries, and hashtags, helping professionals maintain a consistent and engaging online presence.

---

## 🚀 Features

- ✍️ Generate LinkedIn posts using AI
- 🧠 Multi-agent architecture:
  - Content Agent → writes posts
  - Hashtag Agent → generates hashtags
  - Summary Agent → creates summaries
- 📂 Saves generated posts into text files
- ⚡ Simple and extensible design

---

## 🏗️ Architecture

The system is designed using multiple AI agents:

1. **Content Agent**
   - Generates the main LinkedIn post

2. **Hashtag Agent**
   - Suggests relevant hashtags

3. **Summary Agent**
   - Creates a short summary of the post

---

## 🛠️ Tech Stack

- Python
- AI Models (Ollama / OpenRouter / LLMs)
- File-based storage (text files)
